# Tpa3221_classD_Amp
## For 8 Ω speaker
Mode = master <br>
Modulation Mode : AD Mode<br>
Freq = 600khz <br>
Gain = 18dB  ( Vout/Vin = 8 ) <br>
Functional Mode: Stereo BTL <br>
LC Fliter : 10uH + 680nF *(approx 61kHz)* 20kHz -> Gain≈+0.5dB@8Ω<br>
*Recommed Fliter capacitor use Metalized Film(like CCB)*<br><br>
